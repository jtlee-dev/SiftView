# SiftView

A lightweight, cross-platform text and file viewer for inspecting, comparing, and lightly editing text and mixed-format data. **Viewer-first, editor-second.**

## Features

- **Open & view** — JSON, CSV, XML, HTML, YAML, .env, properties, plain text
- **Format** — Pretty-print JSON, align CSV columns, indent XML/YAML, normalize properties
- **Diff** — Compare tabs or clipboard, side-by-side or inline view
- **Save** — Save to disk (desktop) or download (web)
- **Tabs** — Pin, close all, restore closed

## Get SiftView

### Desktop app (recommended)

Download the latest [Release](https://github.com/jtlee-dev/SiftView/releases) for your OS:

- **macOS** — `.dmg` (Apple Silicon or Intel)
- **Windows** — `.msi` installer
- **Linux** — `.deb` or `.AppImage`

### Web demo (no install)

Open **[https://jtlee-dev.github.io/SiftView/](https://jtlee-dev.github.io/SiftView/)** in your browser. Use Open (file picker), Save (download), Format, Diff, and Copy. For full file-system access and native shortcuts, use the desktop app.
